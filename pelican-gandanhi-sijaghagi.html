Pelican 간단히 시작하기

Date Sun 30 July 2017 By SkawnK Category Pelican. Tags python pelican web

Python과 pip가 설치된 환경이라 가정한다.

  • 설치 후 구동하기
    1. pip install pelican Markdown
    2. pelican-quickstart
    3. pelican cotent
    4. cd output
    5. python -m pelican.server
    6. http://localhost:8000

브라우저로 로컬호스트 접속해 "Hello, world!" 격의 Pelican 블로그가 완성된 것을 확인할 수 있다. 이 상태에서는 아티클이 전혀 없는 껍데기 상태만 볼 수 있다.

  • 테마 적용하기
    1. git clone https://github.com/getpelican/pelican-themes.git
    2. pelicanconf.py를 엶.
    3. THEME = 'theme/bootstrap2'를 하단에 추가함.
    4. Pelican 디렉토리 내 theme 디렉토리를 만들고 bootstrap2 테마를 복사해 넣음.
    5. pelcian content

적용된 테마의 코드를 수정하여 자신이 원하는 스타일로 커스터마이징할 수 있다.

태그 클라우드(Tag Cloud)는 Pelican의 기본 기능은 아니며 플러그인 방식으로 구현된다. 태그 클라우드 플러그인 설치 후 그것을 HTML 안에 Jinja로 배치해야 하며, 적지 않은 수의 테마에서는 Jinja 배치 구현이 기본적으로 잡혀 있다.

  • (참고) pelican-plugins_tag_cloud
