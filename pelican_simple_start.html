Pelican 간단히 시작하기

Date 일 30 7월 2017 By SkawnK Category Pelican. Tags python pelican web

Python과 pip가 설치된 환경이라 가정한다.

  • 설치 후 구동하기
    1. pip install pelican Markdown
    2. pelican-quickstart
    3. pelican cotent
    4. cd output
    5. python -m pelican.server
    6. http://localhost:8000

브라우저로 로컬호스트 접속해 "Hello, world!" 격의 Pelican 블로그가 완성된 것을 확인할 수 있다. 이 상태에서는 아티클이 전혀 없는 껍데기 상태만 볼 수 있다.

  • 테마 적용하기
    • pelicanconf.py를 엶.
    • THEME = 'theme/bootstrap2'를 하단에 추가함.
    • Pelican 디렉토리 내 theme 디렉토리를 만들고 bootstrap2 테마를 복사해 넣음.
